# Customer Query Resolver

## Install and Run

To run it locally

````
# In the project repository
$ python3 -m venv venv
$ source venv/bin/activate
$ python -m pip install -r requirements.txt
$ sh entrypoint.sh
````
or you can just start it up with docker

````
$ docker-compose build
$ docker-compose up
````
It will start at 3015 port.
